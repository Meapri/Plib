This directory contains Android/aarch64 prebuilt binaries extracted from Termux packages:

- proot 5.1.107-71 (Package: proot, Architecture: aarch64)
  Source/Homepage: https://proot-me.github.io/
  Package source repository: https://github.com/termux/termux-packages
  Package URL: https://packages.termux.dev/apt/termux-main/pool/main/p/proot/proot_5.1.107-71_aarch64.deb
  Package SHA256: 991e8f5fc9d208b6e9020e6319a42870d764c487cc8fe76fee40b63f1aa2fc9f
  Extracted files used here:
  - usr/bin/proot -> libalr_proot.so
  - usr/libexec/proot/loader -> libproot-loader.so

- libtalloc 2.4.3 (Package: libtalloc, Architecture: aarch64)
  Homepage: https://talloc.samba.org/talloc/doc/html/index.html
  Package source repository: https://github.com/termux/termux-packages
  Package URL: https://packages.termux.dev/apt/termux-main/pool/main/libt/libtalloc/libtalloc_2.4.3_aarch64.deb
  Package SHA256: ac81ad623d74c209718b9f3acb2dd702cc8a88c431e820d212229910b4db29da
  Extracted file used here:
  - usr/lib/libtalloc.so.2.4.3 -> libtalloc.so

Packaging notes:
- proot and its matching loader are kept from the same Termux proot package revision.
- proot is copied as libalr_proot.so so Android APK native library extraction keeps it executable.
- libtalloc.so.2 dependency was patched to libtalloc.so for APK native library naming compatibility.
- RPATH/RUNPATH was patched to $ORIGIN so the executable can resolve libtalloc from nativeLibraryDir.
- Runtime invocations must set PROOT_LOADER=<nativeLibraryDir>/libproot-loader.so and PROOT_TMP_DIR=<cacheDir>/proot-tmp.
